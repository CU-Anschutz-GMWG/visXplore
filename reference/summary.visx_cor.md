# Summary method for visx_cor objects

Displays a formatted correlation/association matrix with significance
indicated by stars.

## Usage

``` r
# S3 method for class 'visx_cor'
summary(object, ...)
```

## Arguments

- object:

  a visx_cor object

- ...:

  additional arguments (ignored)

## Value

a list with formatted matrix (Rnew), row_id, and col_id, invisibly
